让 WebStorm 支持 ECMAScript 6 和 JSX


Oh shit! WebStrom竟然不支持ES6？！
别闹！怎么可能！！！
当然不可能了
经过我一番捣鼓终于发现了在设置里的小秘密…
看下图

默认是 ECMAScript 5.1 的语法支持
要想 WebStrom 的代码提示和静态语法检测支持 ES6 和 JSX 得在 setting 里把 JavaScript language version 改成 JSX harmony
如果你不写 JSX 可以改成 ECMAScript 6，JSX是JS的超集，已经包含了ES6，而且我平常会写点React，所以这里我选了 JSX harmony
好了，改完就过静态语法检测了！！！又可以愉快的打代码了~~
